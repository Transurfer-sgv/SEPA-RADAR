# SEPA RADAR

마크 미너비니 SEPA 방법론 기반 한국 주식(KRX) 일일 모니터링.

- 리포: https://github.com/Transurfer-sgv/SEPA-RADAR
- 대시보드: https://transurfer-sgv.github.io/SEPA-RADAR/

## 무엇을 하나

| 기능 | 내용 |
|---|---|
| 보유종목 점검 | 차트 + SEPA 4단계 + 트렌드 템플릿 8조건 체크리스트 + VCP 상태 |
| 매수타이밍 스크리닝 | 전 종목에서 VCP 피벗 돌파(BREAKOUT)·임박(NEAR) 종목을 매일 자동 추출 |
| 자동 실행 | GitHub Actions가 평일 17:40 KST(장마감 후)에 실행, 결과를 대시보드에 반영 |
| 알림(선택) | 텔레그램 시크릿을 등록한 경우에만 전송, 없으면 자동 생략 |

모든 판정은 **장마감 후 종가(EOD) 기준**입니다.

## 보유종목 바꾸는 법

1. 리포 페이지에서 `config.json` 클릭
2. 오른쪽 연필 아이콘(Edit) 클릭
3. `"holdings"` 목록의 6자리 티커를 수정
4. 초록색 **Commit changes** 버튼

다음 자동 실행부터 반영됩니다.

## 실행 시각 바꾸는 법

`.github/workflows/daily.yml`의 `cron: "40 8 * * 1-5"`를 수정합니다. UTC 기준이므로 KST에서 9시간을 빼야 합니다(예: 19:00 KST → `"0 10 * * 1-5"`).

## 판정 기준 요약

| 항목 | 기준 |
|---|---|
| 트렌드 템플릿 | 주가>150·200일선, 150>200일선, 200일선 1개월 상승, 50>150·200일선, 주가>50일선, 52주 저가 +30% 이상, 52주 고가 -25% 이내, RS≥70 |
| SEPA 단계 | 이평선 배열·200일선 기울기 + 직전 60거래일 구조 이력으로 1~4단계 분류 |
| VCP | 수축 2~6회·깊이 점진 축소·첫 수축 ≤35%·최종 ≤12%·베이스 ≥20일·거래량 드라이업 |
| 돌파(BREAKOUT) | 종가 피벗 상향 돌파 + 거래량 50일 평균 1.5배 이상 |
| 임박(NEAR) | 피벗 -3% 이내 접근 |

## 한계와 주의

1. **EOD 기반**: "오늘 돌파"는 장 마감 후 확정됩니다. 실전에서는 전일 저녁 임박(NEAR) 목록의 피벗 가격을 증권사 MTS 가격알림에 등록해 장중 대응하세요.
2. **RS는 IBD 공식 지표가 아닌** 가중수익률(3·6·9·12개월) 백분위 근사입니다.
3. **단계 분류는 휴리스틱**으로, 경계 구간(2↔3단계) 오분류가 있을 수 있습니다.
4. **pykrx 의존**: KRX 웹 구조 변경 시 수집이 실패할 수 있습니다.
5. 본 도구는 스크리닝 보조이며 매수·매도 추천이 아닙니다.
